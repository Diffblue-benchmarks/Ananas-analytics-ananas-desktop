    Installation

    Ananas Analytics Desktop can run on any OS.

    1. Create an account from https://ananasanalytics.com

    2. Download Ananas Analytics executable from https://ananasanalytics.com

    3. Run the downloaded Ananas Analytics executable

    4. Login with your credential

    5. Enjoy analytics
